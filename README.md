# floofbox
a dropbox like flask script for a linux based nas



# to run
### clone this repo and run (assuming python is installed)
```bash
cd floofbox
```
### and then
```bash
python3 -m venv .venv
```

```bash
source .venv/bin/activate
```

```bash
pip install -r requirments.txt
```

```bash
python setup.py
```
### or

```bash
python3 setup.py
```
